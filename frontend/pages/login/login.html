Recuperar acceso

Esta aplicación funciona sin conexión a internet. Para restablecer una contraseña:

  • Pídele al administrador que reinicie tu contraseña desde Configuración → Usuarios.
  • Si eres el administrador y perdiste el acceso, ejecuta el script scripts/recuperar_admin.sql con psql para restablecer la contraseña a admin123.
